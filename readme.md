# UMLRCWA

This repository contains a fork of the MATLAB RCWA (Rigorous Coupled-Wave Analysis) code developed by Vladimir Podolskiy and the Multiscale Electromagnetics Group at the University of Massachusetts Lowell.

This fork is based on version v0.2.2 of the original code and is not an official release of the Podolskiy group.

## Purpose
Maintained for research use and reproducibility of numerical experiments.  
The numerical implementation follows the original codebase.
